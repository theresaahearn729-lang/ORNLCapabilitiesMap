ORNL Campus Capabilities Explorer

An interactive browser-based dashboard for exploring Oak Ridge National Laboratory campus facilities and research capabilities.

The dashboard includes an interactive campus map with searchable facility labels, capability descriptions, campus-area navigation, building-focused zoom, editable labels, and a full-screen presentation mode.

Features
Interactive ORNL campus map
Search by building, facility, or capability
Click a label to zoom to the building
Pulsing building-location indicator
Editable and movable facility labels
Add or remove custom labels
Campus-area navigation
Off-campus facility information
Full-screen map view for presentations
Self-contained HTML file with no installation required
How to Use
Download the HTML file.
Open it in Chrome, Edge, or Firefox.
Use the search panel or click a facility label.
Select Full Screen Map for presentation use.
Notes

Label edits are stored locally in the browser. To share permanent updates, save or export the edited dashboard and distribute the updated HTML file.
